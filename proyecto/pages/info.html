Acerca del proyecto

Nuestro proyecto permite gestionar tareas de manera sencilla.
Se pueden crear, editar y eliminar tareas, además de filtrarlas por estado.

Funcionalidades

  • Agregar tareas
  • Editar tareas
  • Eliminar tareas
  • Filtrar por estado
  • Ordenar por fecha
  • Contador de tareas

Tecnologías utilizadas

  • HTML
  • CSS
  • JavaScript
  • localStorage
  • Visual Studio Code
  • GitHub
  • Discord
